# Experiment Configuration
# Simple shared data directory approach

# Benefits of Shared Directory:

# ✅ No data duplication (saves space)
# ✅ Consistent data across all experiments
# ✅ Easier management and backup

# Data Sources:
# 1. Automatic download from Zenodo (first run)
# 2. Manual placement in ./cyber/ directory  
# 3. Sample data generation (fallback)

# master-experiment-suite/
# ├── cyber/                  ← Shared data (all experiments)
# ├── batch/results/          ← Batch results
# ├── online/results/         ← Online results  
# ├── cl_case1/results/       ← CL case 1 results
# └── cl_case2/results/       ← CL case 2 results
